# Place google-services.json File

## File Location

Place your `google-services.json` file in:

```
android/app/google-services.json
```

**Important**: 
- The file must be in `android/app/` folder
- NOT in `android/` folder
- The exact path is: `android/app/google-services.json`

---

## Steps

1. **Copy your `google-services.json` file**
2. **Paste it into**: `android/app/google-services.json`
3. **Verify the file exists** in the correct location

---

## Verify File Location

The file structure should look like:

```
eqmsapp/

│   │   ├── google-services.json  ← HERE!

└── ...
```
